# Capacitación IIEE — Sistema Integral de Gestión de Capacitaciones

Aplicación web desarrollada para el seguimiento, análisis y gestión de capacitaciones institucionales del Instituto de Investigaciones Económicas y Estadísticas (IIEE).
La herramienta permite administrar registros de formación, generar indicadores ejecutivos, controlar calidad de datos y sincronizar información con Google Sheets para trabajo colaborativo multiusuario.

# ✨ Funcionalidades principales

## 📊 Dashboard ejecutivo

La aplicación incorpora un tablero interactivo con:

* KPIs automáticos:

  * Horas de capacitación
  * Cantidad de capacitaciones
  * Personas involucradas
  * Relación entre capacitaciones recibidas y otorgadas
  * Evolución temporal
  * Indicadores por equipo

* Visualizaciones dinámicas:

  * Horas por equipo
  * Distribución mensual
  * Capacitaciones por año
  * Personas capacitadas
  * Top capacitadores
  * Top temáticas

* Narrativa automática de gestión del conocimiento:

  * Resumen ejecutivo dinámico
  * Insights institucionales
  * Interpretación de tendencias

* Filtros avanzados:

  * Año
  * Equipo
  * Tipo de capacitación
  * Temática
